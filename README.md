# checkbox-tm
Prototipo web para la gestión y control de entrega de beneficios a trabajadores. Incluye perfiles de Guardia y Recursos Humanos, registro de entregas, bloqueo de doble beneficio, historial, simulación de escaneo QR y gestión de trabajadores por planta (Valparaíso y Casablanca).  Proyecto académico – Ingeniería en Informática.
